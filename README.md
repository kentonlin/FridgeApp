#Fridge
Group Task Manager for families 


##Features 
* Text reminders for daily medications and prescription refills 
  * Iron.io
  * Twilio API 
* Map of nearest pharmacies 
  * Google Maps API
* Display user prescriptions and doctors 
  * React Framework
  * HTML/CSS 
  * Bootstrap 
* Specialist recommender given user symptoms
  * Synaptic.js 
  * BetterDoctor API
